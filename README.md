# Cool Clicker

Cool Clicker is a fun and interactive 3D clicker game built with Java and JavaFX.

## Gameplay

The objective of the game is simple: click the rotating 3D cube to earn points! As your score increases, you can spend points to purchase upgrades that automate point generation or enhance your clicking power.

## Features

- **3D Graphics:** Experience the game with a 3D rotating cube (requires Scene3D support).
- **Manual Clicking:** Click the cube to increase your score manually.
- **Auto Upgrades:** Purchase various auto-clicking upgrades to earn points passively:
  - **Auto Clicker:** A basic auto-clicker.
  - **Grandma:** A classic clicker upgrade inspired by cookie clickers.
  - **Cube Factory:** A factory that produces points.
  - **Hypercube:** The ultimate cube for massive point generation.
- **Click Upgrades:** Improve the number of points you earn per manual click.

## Technologies Used

- Java 22
- JavaFX 22 (Controls, FXML, Media)
- Maven (Dependency Management and Build)

## How to Run

1. Make sure you have Java 22 or higher installed.
2. Clone this repository.
3. Use Maven to build and run the application:
   ```bash
   mvn clean javafx:run
   ```
